**AntiRaidBot** - Discord-бот для защиты серверов от рейдов, крашей, злоупотребления правами и массовых вредоносных действий. Он помогает владельцам серверов быстро реагировать на угрозы, ограничивать опасные роли, контролировать действия администрации и держать безопасность сервера под рукой.

[Добавить AntiRaidBot на сервер](https://discord.com/oauth2/authorize?client_id=1376482498506391673&scope=bot+applications.commands&permissions=8)

## Почему AntiRaidBot

AntiRaidBot создан для серверов, где важны скорость реакции и понятное управление защитой. Бот объединяет автоматическую защиту, ручные emergency-команды, глобальные блокировки и удобные панели настройки через slash-команды и интерактивные компоненты Discord.

Основные преимущества:

- Защита от массового создания и удаления ролей, каналов и других опасных действий.
- Система доверенных пользователей, чтобы не мешать проверенной администрации.
- Лимиты действий для снижения риска краша сервера изнутри.
- Экстренные команды для быстрого реагирования во время рейда.
- Глобальная сеть банов между связанными серверами.
- Удобные v2-компоненты Discord вместо перегруженных текстовых меню.
- Премиум-функции для серверов, которым нужна усиленная защита и кастомизация.

## Основные функции

### Антирейд-защита

AntiRaidBot отслеживает подозрительную активность и помогает остановить пользователей, которые массово создают или удаляют роли и каналы. Владелец сервера может настроить лимиты действий и список доверенных лиц.

Команды:

- `/settings` - главная панель настройки защиты.
- `/limits` - настройка суточных лимитов действий.
- `/trusted` - управление доверенными пользователями.
- `/status` - текущий статус защиты сервера.
- `/raid` - быстро включить или выключить режим защиты.

### Защита ролей

Бот помогает контролировать выдачу опасных ролей и ограничивать доступ к критическим командам.

Команды:

- `/role_blacklist` - список ролей, которые нельзя выдавать.
- `/aban_menu` - роли, которым разрешено использовать `/aban`.
- `/gban_menu` - роли, которым разрешено использовать `/gban`.
- `/creact_menu` - роли, которым разрешено использовать защитные действия.

### Экстренная команда `/aban`

`/aban` - команда для быстрой нейтрализации нарушителя. Она снимает все роли с цели, а также снимает роли с администратора, который использовал команду, чтобы снизить риск злоупотребления.

Команды:

- `/aban @user` - снять все роли с нарушителя.
- `/aban_menu` - настроить роли, которым доступна команда.

### Глобальные баны

AntiRaidBot поддерживает связанную сеть серверов. Если у вас несколько Discord-серверов, вы можете объединить их и применять блокировки сразу по всей сети.

Команды:

- `/gsync` - подключить сервер к глобальной сети.
- `/gban @user` - забанить пользователя на связанных серверах.
- `/gunban @user` - снять глобальный бан.
- `/grole_copy` - скопировать роль на связанные серверы.

### Freeze Mode

Премиум-режим полной заморозки сервера. Он помогает остановить рейд в моменте: блокирует создание и удаление каналов, ролей, вебхуков и добавление новых ботов сторонними лицами.

Команда:

- `/freeze_mode` - включить или выключить режим заморозки.

### Проверка новых ботов

Бот может помогать отслеживать добавление сторонних ботов и реагировать на подозрительные действия.

Команда:

- `/bot_verify` - настройка проверки новых ботов.

### Массовые инструменты модерации

Для владельцев серверов доступны команды для обслуживания сервера и восстановления порядка.

Команды:

- `/role_clear` - удалить выбранную роль у всех участников.
- `/amnesty` - разбанить всех пользователей на сервере.
- `/permcopy` - скопировать права одного канала в другой.

### Профиль и информация

AntiRaidBot также содержит удобные информационные команды для пользователей и владельцев серверов.

Команды:

- `/help` - интерактивная справка по боту.
- `/ping` - проверить задержку и состояние бота.
- `/profile` - профиль пользователя.
- `/invite` - ссылка для добавления бота.

## Премиум-возможности

Премиум открывает расширенные функции для серверов, которым нужна более сильная защита:

- `/freeze_mode` - полная заморозка сервера во время атаки.
- `/custom` - кастомная аватарка и баннер AntiRaidBot на вашем сервере.
- Увеличенные лимиты списков ролей: до 10 ролей вместо стандартных 5.
- Визуальный премиум-статус в профиле.

## Для кого этот бот

AntiRaidBot подойдёт:

- крупным Discord-серверам;
- игровым сообществам;
- проектам с активной администрацией;
- серверам, где важно быстро реагировать на рейды;
- владельцам нескольких серверов, которым нужна единая сеть глобальных банов.

## Быстрый старт

1. Добавьте бота на сервер: [пригласить AntiRaidBot](https://discord.com/oauth2/authorize?client_id=1376482498506391673&scope=bot+applications.commands&permissions=8).
2. Выполните `/settings`, чтобы открыть панель защиты.
3. Настройте `/limits` и `/trusted`.
4. Добавьте опасные роли в `/role_blacklist`.
5. Настройте доступ к `/aban` и `/gban` через меню ролей.
6. Для нескольких серверов используйте `/gsync`.
